# 기본 투자 체크리스트

## 실적

- [ ] 매출 성장률이 개선되고 있는가?
- [ ] 영업이익률(OPM)이 개선되고 있는가?
- [ ] 컨센서스를 상회했는가?
- [ ] 일회성 이익/비용은 없는가?

## 재무 안정성

- [ ] 부채비율은 부담스럽지 않은가?
- [ ] 현금흐름이 양호한가?
- [ ] FCF가 개선되고 있는가?

## 밸류에이션

- [ ] PER이 과거 평균 대비 낮은가?
- [ ] PBR이 ROE 대비 합리적인가?
- [ ] 동종업계 대비 저평가인가?

## 모멘텀

- [ ] 실적 상향 조정 가능성이 있는가?
- [ ] 신규 수주/제품/고객사 모멘텀이 있는가?
- [ ] 산업 트렌드와 맞물려 있는가?

## 리스크

- [ ] 투자포인트가 훼손될 리스크는 무엇인가?
- [ ] 원가, 환율, 금리, 규제 리스크는 어떤가?
- [ ] 경쟁 심화 가능성은 있는가?
