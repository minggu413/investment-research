# 매주 반복 리서치 작업 목록

이 문서는 GitHub Actions로 자동화할 주간 리서치 루틴 초안입니다.
처음에는 결과 파일을 자동 생성하는 구조만 만들고, 이후 DART, 뉴스, PDF, 재무데이터 수집 코드를 하나씩 연결합니다.

## 실행 일정 초안

- 실행 주기: 매주 월요일 오전 8시 30분 KST
- 목적: 주말 동안 발생한 뉴스, 공시, 리포트, 수급 변화를 정리하고 이번 주 확인할 종목을 추립니다.
- 결과 저장 위치: `리포트요약/주간리서치/`

## 주간 리서치 작업 순서

| 순서 | 작업 | 산출물 | 저장 위치 |
|---:|---|---|---|
| 1 | 관심종목 목록 불러오기 | 이번 주 점검 대상 종목 | `리포트요약/주간리서치/관심종목.md` |
| 2 | 최근 뉴스 확인 | 종목별 주요 뉴스 3~5개 | `리포트요약/주간리서치/뉴스요약_YYYY-MM-DD.md` |
| 3 | DART 공시 확인 | 신규 공시와 투자 영향 | `리포트요약/주간리서치/DART공시_YYYY-MM-DD.md` |
| 4 | 증권사 리포트 확인 | 목표가, 투자의견, 실적 추정 변화 | `리포트요약/주간리서치/리포트요약_YYYY-MM-DD.md` |
| 5 | 실적/컨센서스 변화 점검 | 매출, 영업이익, OPM, ROE, FCF 변화 | `재무데이터/주간_핵심지표_YYYY-MM-DD.csv` |
| 6 | 동종업계 비교 업데이트 | 비교표 변경점 | `동종업계비교/주간비교_YYYY-MM-DD.md` |
| 7 | 투자 체크리스트 업데이트 | 통과/보류/탈락 판단 | `투자체크리스트/주간점검_YYYY-MM-DD.md` |
| 8 | 이번 주 액션 정리 | 추가 확인, 보유, 관망, 축소 후보 | `리포트요약/주간리서치/주간액션_YYYY-MM-DD.md` |

## 우선 자동화할 작업

1. 관심종목 목록 읽기
2. 주간 리서치 결과 파일 생성
3. DART 공시 수집
4. 뉴스 요약
5. PDF 리포트 요약
6. 재무지표 업데이트
7. GitHub Issue로 확인할 질문 생성

## 주의사항

- 이 자동화는 투자 판단을 대신하지 않습니다.
- 자동화 결과는 원문 공시, IR 자료, 리포트, 뉴스와 교차 확인합니다.
- 매수/매도 판단은 투자자가 직접 결정합니다.
