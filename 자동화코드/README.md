# 자동화코드

이 폴더는 반복 리서치 작업을 자동화하는 코드와 실행 방법을 보관합니다.

## 앞으로 만들 수 있는 자동화

- DART 공시 수집
- 증권사 PDF 리포트 요약
- 재무제표 핵심 지표 계산
- 동종업계 비교표 생성
- 투자 체크리스트 자동 업데이트
- 보고서 초안 생성

처음에는 템플릿과 수동 정리부터 시작하고, 반복되는 작업이 명확해지면 자동화 코드를 추가합니다.

## 주간 리서치 자동화 초안

- 작업 목록 문서: `자동화코드/매주_리서치_작업목록.md`
- GitHub Actions 초안: `.github/workflows/weekly-research.yml`
- 실행 일정 초안: 매주 월요일 오전 8시 30분 KST
- 결과 저장 위치: `리포트요약/주간리서치/`

현재 워크플로는 실제 뉴스/DART/PDF 수집 전 단계입니다. 먼저 주간 리서치 초안 파일을 자동 생성하고, 이후 데이터 수집 코드를 하나씩 연결하는 구조입니다.
