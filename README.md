# 투자업무 작업공간

이 폴더는 투자 리서치 자료, 보고서 초안, 시장뉴스, 템플릿, 자동화 작업을 모아두는 최상위 작업공간입니다.

핵심 원칙은 다음과 같습니다.

> 판단은 투자자가 하고, 반복 정리 작업은 Codex가 돕는다.

## 현재 기준 위치

```text
C:\Users\lmg04\Documents\Codex\투자업무
```

이 폴더 자체가 GitHub 저장소와 연결된 기준 작업공간입니다. 예전의 `investment-research` 폴더 안에 있던 파일과 폴더는 이 위치로 꺼내 정리했습니다.

## 주요 폴더

| 폴더 | 용도 |
|---|---|
| `종목 보고서` | 기업별 투자보고서와 종목 분석 보고서 |
| `종목 모니터링` | 보유/관심 종목의 추적 메모 |
| `시장뉴스` | 시장뉴스 취합 결과 |
| `산업 보고서` | 산업, 테마, 기술 트렌드 보고서 |
| `투자종목 발굴` | 투자 아이디어, 스크리닝, 주도주 후보 정리 |
| `국민연금 지분공시` | 국민연금 지분공시 관련 자료 |
| `동종업계비교` | 경쟁사와 업종 비교표 |
| `재무데이터` | 핵심 재무지표와 계산용 데이터 |
| `투자체크리스트` | 투자 전 점검 항목 |
| `자동화코드` | 반복 리서치 작업 자동화 관련 파일 |
| `기타 보고서` | 분류가 아직 정해지지 않은 보고서 |

## 템플릿 파일

반복 요청에 쓰는 템플릿은 현재 이 폴더 최상위에 둡니다.

- `기업분석 보고서 템플릿.md`
- `종목보고서_요약_템플릿.md`
- `시장뉴스 취합 템플릿.md`
- `산업보고서_요약_템플릿.md`
- `투자종목_발굴_네러티브_템플릿.md`
- `투자종목_발굴_주도주_템플릿.md`

`_investment-research`가 붙은 파일은 예전 GitHub 저장소 안에서 꺼낸 동일 계열 템플릿입니다. 바로 삭제하지 않고 비교용으로 보존했습니다.

## 자주 쓰는 작업 요청

### 종목 보고서 업데이트

```text
작업 위치:
C:\Users\lmg04\Documents\Codex\투자업무

종목 보고서\현대모비스.md를 최신 자료 기준으로 업데이트해줘.
투자포인트, 실적 모멘텀, 밸류에이션, 리스크, 확인할 질문을 중심으로 정리해줘.
```

### 시장뉴스 취합

```text
C:\Users\lmg04\Documents\Codex\투자업무\시장뉴스 취합 템플릿.md를 기준으로
오늘 시장뉴스를 취합해서 시장뉴스 폴더에 저장해줘.
```

### PDF 리포트 요약

```text
첨부한 PDF 리포트를 종목보고서_요약_템플릿.md 기준으로 요약해줘.
투자포인트, 리스크, 실적 추정, 밸류에이션, 추가 확인 질문을 정리해줘.
```

## 정리 기준

- 기업별 보고서: `종목 보고서`
- 산업/테마 자료: `산업 보고서`
- 뉴스 취합 결과: `시장뉴스`
- 자동화 관련 파일: `자동화코드`
- 비교표: `동종업계비교`
- 재무표와 지표: `재무데이터`
- 판단 체크리스트: `투자체크리스트`
