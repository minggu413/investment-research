# 투자 리서치 저장소

모멘텀 가치투자를 위한 개인 리서치 저장소입니다.

## 목적

- 기업 분석 보고서 정리
- 투자 체크리스트 관리
- PDF 리포트 요약
- 재무제표 자동 정리
- 동종업계 비교표 작성
- 투자 아이디어 스크리닝
- 반복 리서치 작업 자동화

## 주요 폴더

| 폴더 | 용도 |
|---|---|
| `기업보고서` | 기업별 투자보고서 보관 |
| `보고서템플릿` | 반복해서 쓰는 보고서 양식 |
| `투자체크리스트` | 투자 판단 전 점검 항목 |
| `재무데이터` | 매출, 영업이익, OPM, ROE, FCF 등 데이터 |
| `동종업계비교` | 경쟁사 비교표 |
| `산업메모` | 산업, 테마, 기술 트렌드 메모 |
| `리포트요약` | 증권사 PDF, IR 자료, 뉴스 요약 |
| `자동화코드` | 자료 수집, 요약, 표 정리 자동화 코드 |

## 기본 원칙

판단은 투자자가 하고, 반복 작업은 Codex가 돕습니다.

## 사용 예시

- `기업보고서/현대모비스.md`에 기업별 투자보고서를 작성합니다.
- `보고서템플릿/기업분석_템플릿.md`을 복사해서 새 보고서를 시작합니다.
- `투자체크리스트/기본_투자체크리스트.md`로 투자 전 점검합니다.
- `동종업계비교/자동차부품_비교표.md`처럼 업종별 비교표를 만듭니다.
