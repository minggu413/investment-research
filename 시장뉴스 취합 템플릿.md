# 시장뉴스 취합 요청사항

## 가이드라인
- 제목은 굵은 글씨로 작성함.
- 요약 내용은 음슴체로 작성함.
- URL은 링크 텍스트로 바꾸지 않고 주소 그대로 작성함.
- 요약 내용은 3줄로 작성함.
- 투자자 해석은 2줄로 작성함. 음슴체.
- 기사와 기사 사이에는 빈 줄 2줄을 넣음.
- 기사 앞에 번호를 붙이지 않음.
- 같은 기사나 같은 내용이 중복되지 않도록 함.
- MD파일: 사용자가 별도로 요청할 경우에만 생성함.
- 국내, 해외 기사 골고루 배분함.
## 기본 요청 문장
```text
시장뉴스 취합.
```
## 카테고리 작성 규칙
- 뉴스 취합 시 관련 기사들을 카테고리별로 묶어서 작성함.
- 카테고리 제목은 `## 국내`, `## 해외`처럼 2단계 제목으로 작성함.
- 사용자가 특정 카테고리를 지정하면 해당 카테고리를 우선함.
- 기사 수 배분은 중요도와 최신성을 기준으로 유동적으로 조정함.
- 적합한 기사가 적은 카테고리는 생략할 수 있음.
## 기본 출력 방식
- 기본 출력 위치: 채팅창
- MD 파일: 사용자가 요청할 때만 생성
## 기본 기사 수
- 기본 요청 기사 수: 40개
- 사용자가 기사 수를 별도로 말하면 해당 수량을 우선함.
