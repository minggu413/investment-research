## 기본 명령어

```text
이벤트 리뷰: 법인 탭 보유종목 중 리뷰 필요 종목, 최근 30일, 음슴체
```

## 위치

```text
https://docs.google.com/spreadsheets/d/19OVo9nFjtHRr_9w8ICzL5JWFXHKJ-sHU2kvLDtT1Ca4/edit?gid=1832700276#gid=1832700276
```

## 대상 선택

```text
대상:
- 법인 탭 보유종목 중 리뷰 필요 종목 (디폴트)
- 별도로 언급시, 법인 탭 보유종목 전체
- 종목명: SK하이닉스, 유니드, 비에이치
```

## 기간 선택

```text
기간:
- 최근 30일(디폴트)
- 최근 7일
- 실적 발표 이후
- 특정 기간: 2026-04-01 ~ 2026-05-10
```

## 반복 명령어

### 1. 최근 뉴스

```text
최근 뉴스: 법인 탭 보유종목 중 리뷰 필요 종목, 최근 30일
```

작업 내용:
- 뉴스/리포트/공시 확인
- 날짜와 제목 굵게 표시
- URL 주소 그대로 표시
- 요약 3줄 음슴체
- 투자자 코멘트 2줄 음슴체

출력 형식:

```text
**2026-01-01  제목**
URL

요약 3줄 음슴체
요약 3줄 음슴체
요약 3줄 음슴체

=> 투자자 코멘트
코멘트 1줄 음슴체
코멘트 2줄 음슴체
```

### 2. 이벤트 리뷰

```text
이벤트 리뷰: 법인 탭 보유종목 중 리뷰 필요 종목, 최근 30일, 음슴체
```

확인 항목:
1. 이벤트 요약
2. 주가/거래량 변화
3. 원인 분석
4. 최근 리포트/뉴스/공시
5. 실적 및 밸류에이션 영향
6. 투자포인트 3가지
7. 리스크 3가지
8. 체크리스트
9. 투자자 판단 메모

출력 형식:

```text
**종목명**

1. 이벤트 요약
- 내용 음슴체

2. 주가/거래량 변화
- 내용 음슴체

3. 원인 분석
- 내용 음슴체

4. 최근 리포트/뉴스/공시
- **2026-01-01  제목**
- URL

5. 실적 및 밸류에이션 영향
- 내용 음슴체

6. 투자포인트 3가지
- 내용 음슴체

7. 리스크 3가지
- 내용 음슴체

8. 체크리스트
- 실적 방향: 통과/보류/탈락
- 모멘텀: 통과/보류/탈락
- 밸류에이션: 통과/보류/탈락
- 리스크: 통과/보류/탈락
- 수급/거래량: 통과/보류/탈락

9. 투자자 판단 메모
- 보유 / 추가매수 / 축소 / 매도 / 관망
- 이유 음슴체
```

### 3. 종목 점검

```text
종목 점검: 유니드, 최근 30일, 음슴체
```

작업 내용:
- 최근 뉴스/리포트/공시 확인
- 주가/거래량 변화 확인
- 실적 및 밸류에이션 영향 정리
- 투자포인트와 리스크 재정리
- 보유 / 추가매수 / 축소 / 매도 / 관망 중 판단 메모 작성

## 리뷰 필요 종목 기준

```text
다음 조건 중 하나라도 해당하면 리뷰 필요 종목으로 분류:
- 손실률 -10% 이하
- 1개월 수익률 음수
- 비중 5% 초과
- 실적 컨센서스 하향
- 최근 뉴스가 기대감 중심이고 숫자 확인 부족
- 거래량 급증 또는 급감
- 공시/리포트 발생
```

## 판단 기준

```text
추가매수:
- 실적 방향 개선
- 모멘텀 유지
- 밸류에이션 부담 낮음
- 리스크 통제 가능

보유:
- 기존 투자포인트 유지
- 실적/뉴스 흐름 중립 이상
- 단기 조정이나 훼손 근거 부족

관망:
- 뉴스는 있으나 숫자 확인 부족
- 실적 영향 불명확
- 주가 변동성 과도

축소:
- 투자포인트 약화
- 실적 하향 또는 모멘텀 훼손
- 비중 과도

매도:
- 핵심 투자논리 훼손
- 손실 확대에도 회복 근거 부족
- 리스크가 기대수익보다 큼
```
