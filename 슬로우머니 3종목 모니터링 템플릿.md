- 최근 기업 관련 소식, 업종 관련 소식, 해당 종목에 대한 영향까지 정리

- 기본 출력 위치: MD 파일

## 실행 명령어
```text
종목
```

## 실행 순서
1. Codex가 먼저 사용자에게 아래 질문을 함.
2. 사용자가 종목명 또는 종목코드 2개를 답변함.
3. Codex는 해당 종목에 대해 아래 항목을 조사함.
4. 결과를 채팅창에 템플릿 형식으로 출력함.

## 첫 질문
```text
모니터링할 3종목 알려줘.
종목명 또는 종목코드 3개만 입력하면 됨.
```

## 조사 항목 - 3종목 각각에 대해
- 최근 종목 관련 뉴스 1개
- 최근 업종 관련 뉴스 1개
- 각 뉴스의 핵심 요약 3줄
- 해당 종목에 대한 영향 2줄

## 조사 기준
- 종목 뉴스:
  - 해당 기업 자체와 직접 관련 있는 뉴스 우선
  - 실적, 수주, 신제품, 임상, 계약, 규제, 지배구조, 배당, 자사주, 목표주가 변화 등
  - 단순 시장 기사, 비관련 기사, 이름만 스친 기사 배제
- 업종 뉴스:
  - 해당 종목이 속한 업종 전반의 최근 이슈 1개 확인
  - 업황, 가격, 수요, 정책, 경쟁, 공급망, 수출입, CAPEX, 규제 변화 등
- 영향 판단:
  - 단기 수급 영향인지
  - 중기 실적 영향인지
  - 밸류에이션 재평가 요인인지
  - 일회성 뉴스인지 구분
