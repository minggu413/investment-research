# 슬로우머니 포트폴리오 모니터링 템플릿

## 목적

포트폴리오 전체를 점검하고, 리뷰가 필요한 종목을 선별하는 템플릿.

개별 종목 상세 분석은 `슬로우머니 종목 모니터링 템플릿.md` 사용.

## 기본 명령어

```text
슬로우머니 포트폴리오 모니터링
```

## 기본 위치

```text
https://docs.google.com/spreadsheets/d/19OVo9nFjtHRr_9w8ICzL5JWFXHKJ-sHU2kvLDtT1Ca4/edit?gid=1832700276#gid=1832700276
```

## 기본 조건

```text
대상:
- 법인 탭 보유종목 전체

기간:
- 최근 30일

문체:
- 음슴체
```

## 확인 항목

1. 포트폴리오 요약
2. 수익률 상위/하위 종목
3. 비중 상위 종목
4. 업종/테마 쏠림
5. 현금 비중
6. 최근 1개월 모멘텀
7. 리뷰 필요 종목 선정
8. 다음 액션

## 리뷰 필요 종목 기준

```text
다음 조건 중 하나라도 해당하면 리뷰 필요 종목으로 분류:
- 손실률 -10% 이하
- 1개월 수익률 음수
- 비중 5% 초과
- 최근 실적/뉴스/공시 발생
- 실적 컨센서스 하향
- 최근 뉴스가 기대감 중심이고 숫자 확인 부족
- 거래량 급증 또는 급감
- 포트폴리오 수익 기여도가 큰 종목
```

## 출력 형식

```text
**슬로우머니 포트폴리오 모니터링**

1. 포트폴리오 요약
- 총평가금액:
- 총수익률:
- 현금비중:
- 핵심 특징:

2. 장점
- 내용 음슴체

3. 약점
- 내용 음슴체

4. 업종/테마 점검
- 반도체:
- 바이오:
- 에너지/전력:
- 기타:

5. 리뷰 필요 종목
| 종목 | 이유 | 비중 | 수익률 | 1개월 | 우선순위 |
|---|---|---:|---:|---:|---|

6. 우선 모니터링 대상
- 1순위:
- 2순위:
- 3순위:

7. 다음 액션
- 내용 음슴체
```
